# Student Management System (Python)

一个简单的学生管理系统，使用 Python + Pandas 实现。

## 功能

- 查看所有学生
- 添加学生
- 删除学生
- 查询学生
- CSV 文件数据存储

## 项目结构

```
student-management-python
│
├── data
│   └── students.csv
│
├── src
│   └── main.py
│
└── README.md
```

## 运行方式

进入 src 目录：

```
cd src
```

运行程序：

```
python3 main.py
```

## 示例界面

```
===== 学生管理系统 =====
1 查看所有学生
2 添加学生
3 删除学生
4 查询学生
5 退出
```
